# Tag-der-Demokratie-Specs

Spezifikationen für das Fundraising-Tool rund um den **Tag der Demokratie**.

## Inhalt

- [`SPECS.md`](./SPECS.md) – Fachliche und technische Spezifikation inkl. Datenschema

## Ziel

Dieses Repository beschreibt die Anforderungen an ein Spendensammler-Tool:

- Kampagnen mit Zielbetrag anlegen und verwalten
- Spenden via PayPal, Wero und Kreditkarte (Stripe) entgegennehmen
- Fortschritt in Echtzeit anzeigen (Wikipedia-Stil)
- Kampagnen per QR-Code und Link verteilen
- Spendenbestätigungen und Zuwendungsbestätigungen (≥ 100 €) per E-Mail versenden
- Buchhaltungsübersicht und CSV-Export im Adminbereich
